Overview

Here you will find the source separated audio data, and the subjective evaluation data of that audio from my thesis. For more information on this data, please refer to my thesis document.

Separated Audio

For these experiments, I used a 30-song subset of the MUSDB18 dataset for music source separation. I source separated each song using these five source separation models:
  1. Hybrid Demucs [1]
  2. Demucs-Extra [2]
  3. D3Net [3]
  4. Spleeter [4]
  5. Source SeparationWavenet [5]
